# lms图书管理系统

### 项目依赖的安装

```
npm install
```

### 运行项目

```
npm run serve
```

### 打包项目

```
npm run build
```

### 项目文件

`lms_api` : 后端php代码

`lms.sql` : 数据库文件

`/src` : 前端部分代码

### 自定义配置

See [Configuration Reference](https://cli.vuejs.org/config/).
